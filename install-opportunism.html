Opportunism

The installation of the Fogbow Opportunism is very simple, it uses the Distutils from python to install the module, so all you need to do is get the latest code and run the setup.py and all packages and script will allocated to the right place

Run the following line in your shell

git clone https://github.com/fogbow/fogbow-opportunism.git
cd fogbow-opportunism
sudo python setup.py

After run these commands the installation process will install powernap files, action scripts and will set the init script to start the deamon in the resource initialization.
